


## Syntax

    - php tag <?php code ?>
    - short tag <? code ?>
    - printing tag <? code ?>
    - printing short tag <?= code ?>             // run and print // modern tag
    - flow tags:                                 // if, while, swich                 
        <? control: ?>
        ...
        <? endcontrol ?>

    - instruction1; instruction2; ...
    - print() - functions
    - echo - instruction
